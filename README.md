# minirobo2026-esp.inoについて
他Readmeに記載のあった無線通信用マイコンのコードです。
esp-wroom-32を用いて、PCのreceiverからのUDP通信を受け取りI2C通信を用いてminirobo2026.1が書き込まれたArduinoNanoにデータを送信します。  
# minirobo2026.01_v0.2.inoについて
esp-wroom-32からI2C通信で受信したデータをデシリアライズし、必要なデータを取り出し残りのデータをUART通信でminirobo2026.2が書き込まれたArduinoNanoにデータを送信します。  
タイヤ駆動部分のモーターを制御しており、モーター制御部分のプログラムについてはもう一人の情報担当が担当しました。
# minirobo2026.02_v0.2.inoについて
minirobo.01が書き込まれたArduinoNanoからUART通信で受信したデータをデシリアライズし、モーターを制御しています。
機構部分のモーターを制御しており、モーター制御部分のプログラムについてはminirobo2026.01と同じくもう一人の情報担当が担当しました。
# I2C通信とUART通信両方を用いている点について
元々はもう一人の情報担当の勉強も兼ねて2つの通信を行って貰う予定だったのですが、本番までに終わらせる見通しが立たなかったため通信部分は急遽自分が書くという形になりました。  
その時点では配線と基盤が既に作成されていたため、別々の通信方法で通信するという状況になりました。
